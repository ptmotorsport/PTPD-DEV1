# PT Motorsport PDM System

Professional Power Distribution Module (PDM) system for motorsport applications, featuring an Arduino UNO R4 Minima-based embedded controller with CAN bus integration and comprehensive desktop management software.

**Hardware**: 4-channel intelligent power distribution with temperature monitoring, over-current protection, and dual NeoPixel status displays. CAN bus telemetry and keypad control support.

**Software**: Python-based GUI for real-time monitoring, configuration management, and firmware updates via USB serial interface.

**Key Features**: Live diagnostics, automated protection systems, hardware watchdog, configurable thresholds, and professional operator interface.
